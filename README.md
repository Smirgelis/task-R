# task-R

## Installing
Install Cypress for Mac, Linux, or Windows, then [get started](https://on.cypress.io/install).

```bash
  npm install cypress --save-dev
```
or
```bash
  yarn add cypress --dev
```
or
```bash
  pnpm add cypress --save-dev
```

## Running cypress

```bash
  npx cypress open
```
or
```bash
  yarn cypress open
```
or
```bash
  pnpm cypress open
```
